### Šta je ovo?
Ove je biblioteka za proveru državnih praznika Republike Srbije.

---

### Instalacija

#### NPM

    $ npm install -g praznici
    
#### Yarn

    $ yarn global add praznici

---

### Primeri korišćenja

```
import { isNationalHoliday } from 'praznici';

isNationalHoliday('2021-11-11'); // true
```
